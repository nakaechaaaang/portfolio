# AI×フリーランス サービス販売パッケージ

298,000円のオーダーメードAI参入プランを、メルマガで販売するための全資料一式です。

## サービス概要

- **対象**: フリーランスになりたい × AI を使いたい × でも方法が分からない人
- **提供**: 経営者ジャンルへ参入するためのオーダーメードAIプラン
- **価格**: 298,000円
- **集客**: YouTube / Instagram / TikTok
- **獲得**: PDF プレゼント → メルマガ → 個別相談 → 成約

## ファネル全体図

```
ショート動画(YT/IG/TT)
        ↓
プロフィール(全媒体共通)
        ↓
LP(縦長1ページ)
        ↓
PDF特典 登録
        ↓
メルマガ 7日間教育
        ↓
無料個別相談(ZOOM 60分)
        ↓
成約 298,000円
```

## ローンチまでの推奨順序

1. **PDF を完成** (01) → Canvaでデザイン → PDFエクスポート
2. **LP を作成** (02) → STUDIO / ペライチで公開
3. **メルマガ配信ツール契約** → ConvertKit / オートビズ等 → 7通登録 (03)
4. **プロフィール整備** (05) → 全媒体統一
5. **ショート動画 30本収録** (04) → 各媒体に投稿開始
6. **相談スクリプト練習** (06) → 録画で1回ロープレ
7. **カレンダー開始** (07) → 毎日運用へ

## KPI目標

- リスト1人獲得単価: 500〜2,000円
- メルマガ → 相談予約: 5〜15%
- 相談 → 成約: 20〜40%
- 月10成約 (月298万円) を狙うなら 月500〜1,000リスト
