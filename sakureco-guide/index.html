録ったら、3秒で共有リンク。

サクレコ（SAKU-RECO）は、画面とカメラをブラウザだけで録画して、その場でURLを発行できるツールです。時間無制限・高画質・インストール不要。送られた相手はアプリなしでそのまま再生できます。

What is it

サクレコでできること

「Loomの5分制限が不満」「画質を落としたくない」「アップロードの処理待ちが嫌」——そんな声に応えた、画面録画＆即共有ツールです。

⏱️
時間無制限5分制限なし。2時間級の長い会議・セミナーも1本で録れます。
🔗
3秒で共有リンク停止した瞬間にプレビュー、リンク発行は数秒。処理待ちゼロ。
🖥️
3つの録画モード「画面のみ」「画面＋カメラ（丸ワイプ）」「カメラのみ」から選べます。
✂️
無劣化トリム言い間違えた冒頭・最後をカットして共有。再エンコードなし。
⏩
倍速視聴受け取った人は最大2倍速で視聴できます（ログイン不要）。
📱
スマホでも視聴OK視聴はスマホ含むほぼ全ブラウザ対応（録画形式により一部例外）。
🔑

録画には「合言葉」が必要です（サクレコは関係者専用の録画ツール）。共有された動画リンクの視聴に合言葉は不要で、誰でもそのまま再生できます。

Before you start

はじめる前に

  • 録画は Chrome または Edge を使ってください（推奨ブラウザ）。
  • 録画ページを開くと合言葉を聞かれます（関係者専用）。入力して「入室する」。
  • Mac で「画面の音」を入れたい場合は、Chromeで“タブ”を共有したときのみ取り込めます。
  • 録画はあなたのPC内で処理され、「共有リンクを発行」したときだけクラウドにアップロードされます。

How to record

録画のしかた（ステップ）

1

合言葉を入れて入室

sakureco.surge.sh を Chrome / Edge で開き、合言葉を入力して「入室する」。一度入ればそのブラウザでは記憶されます。

2

録画モードを選ぶ

画面のみ ／ 画面＋カメラ（丸ワイプ） ／ カメラのみ の3択。カメラを選ぶと映りを確認するプレビューが出ます。丸ワイプは画面の左下に自分の顔が丸く合成されます。

3

音声を選ぶ

マイク（自分の声）は初期ON。画面の音（タブ／システム音声）も入れたいときはチェックします。

4

画質を選ぶ

標準（2時間級の長尺もOK）か 高画質（30分以内の短尺向け）。迷ったら標準でOKです。

5

「REC」を押す → 3秒カウントダウン

赤いRECボタンを押すと、ブラウザが「どれを共有しますか？」と聞いてきます（画面全体／ウィンドウ／タブ）。選ぶと3秒カウントダウン後に録画開始です。

6

録画中（一時停止・ワイプ移動）

タイマーが進みます。一時停止でいつでも中断・再開できます。画面＋カメラのときは、プレビュー（または常に最前面の小窓）をドラッグして丸ワイプの位置を移動できます。

7

「■ 停止して確認」

終わったら停止。その場でプレビューが表示され、タイトルを入力できます。さらに無劣化トリムのスライダーで、前後の不要な部分をカットできます（画質劣化なし）。

8

「共有リンクを発行」

ボタンを押すとアップロードが始まり、共有リンクが発行＆自動コピーされます。あとは貼って送るだけ。手元に残したいときは「PCに保存」、録り直すときは「撮り直す」。

🔒

社外秘・個人情報を含む動画は、クラウドに上げず「PCに保存」でのローカル共有がおすすめです。

Share & watch

共有・視聴のしかた

共有リンクを開くと、視聴ページで動画がそのまま再生されます。受け取った人はログイン不要です。

  • 倍速視聴に対応（1× / 1.25× / 1.5× / 1.75× / 2×）。
  • ページからリンクのコピー・ダウンロードができます。
  • 再生できないときは「ダウンロード」で保存してから再生してください。

必要な部分だけ見せる（無劣化トリム）

録画後のスライダーで開始・終了を決めると、その範囲だけを再生する共有リンクになります。元動画はそのまま・再エンコードなし・処理待ちゼロ。共有リンクが watch.html?...&s=開始秒&e=終了秒 の形で、視聴ページが範囲を自動で適用します。

Quality

画質と長さの目安

画質向いている用途ビットレート
標準2時間級の長尺・手順説明・打ち合わせ（容量と安定性◎）画面 約2.5Mbps（2時間≈2.3GB）
高画質30分以内の短尺・細部までくっきり見せたいとき画面 約8Mbps

長尺は標準推奨。録画形式は端末対応に応じて mp4 を優先（非対応時は webm）で保存されます。

Troubleshooting

うまくいかないとき

Q. 合言葉が通らない／「通信エラー」と出る

合言葉が違うと「合言葉が違います」、ネットワーク不調だと「通信エラー」と表示されます。入力を確認し、電波状況を整えて再度お試しください。

Q. 録画ボタンが反応しない／画面が選べない

ブラウザを Chrome か Edge にしてください。一部のブラウザは画面録画に対応していません。

Q. 「画面の音」が入らない（Mac）

Mac では、Chromeで“タブ”を共有したときのみ画面の音を取り込めます。画面全体・ウィンドウ共有だと音が入らないことがあります。

Q. iPhone で再生できない

webm 形式は iPhone の Safari で再生できないことがあります。録画は mp4 対応ブラウザ（Chrome / Edge）で行うとスマホでも見られます。

Q. 動画が途中で止まる・読み込まない

「Claude in Chrome」などの拡張機能が動いていると、再生がストールすることがあります（サクレコ側の不具合ではありません）。拡張をオフにするか、別のブラウザ／「ダウンロード」で保存して再生してください。

Q. 「認証が切れました」と出た

合言葉を入れ直してから、もう一度「共有リンクを発行」してください。録画データは消えていません。

Setup（上級）

自分用にサクレコを構築する（設定方法）

サクレコは フロント（index.html＝録画 / watch.html＝視聴）＋ Cloudflare Worker ＋ R2ストレージ でできています。自分用に立てる手順をまとめます。

🧩

仕組み：ブラウザで録画 → 64MB分割で Worker にアップロード → R2 に保存。視聴は Worker が Range対応（206）で配信。録画ページは合言葉を Worker の /auth で検証します。

手順（ゼロから）

1

Cloudflareにサインアップ（無料）

dash.cloudflare.com/sign-up でメールアドレスとパスワードを入力 → 確認メールのリンクで認証。ドメイン購入は不要です。

2

R2を有効化してバケットを作る

ダッシュボード左の 「R2」 →「R2を始める（Purchase R2 Plan）」→ 「Create bucket」でバケット作成（名前は任意。例：sakureco）。R2は10GB無料・配信転送量も無料です。

💳

R2の有効化時にクレジットカード登録を求められます。無料枠（10GB・転送量0円）の範囲なら請求は発生しませんが、カード登録自体は必須です。

3

操作ツール（wrangler）を準備

Node.js（nodejs.org のLTS）を入れて、ターミナルで：

npm install -g wrangler
wrangler login（ブラウザが開くので「Allow」）

4

Worker（保存・配信＋合言葉認証）をデプロイ

R2バケットをバインドした Worker を wrangler deploy で公開します。Worker は次のエンドポイントを実装します（フロントが呼びます）：

  • POST /auth … 合言葉（X-Sakureco-Key ヘッダ）を検証
  • PUT /put?key=… … 64MB以下の一括アップロード
  • POST /mpu/create / PUT /mpu/part / POST /mpu/complete … 大容量の分割アップロード
  • GET /file/rec/… … Range対応で動画を配信

デプロイ後の配信URLは https://<あなたのWorker>.workers.dev/file/rec/… の形です。

5

フロントに自分のWorker URLを設定（2か所）

自分のWorkerに合わせて、次の2行だけ書き換えます：

  • index.html … const STORAGE_API = 'https://<あなたのWorker>.workers.dev';
  • watch.html … const PUB_HOST = '<あなたのWorker>.workers.dev';

視聴ページは「このホスト＋ /file/rec/ 以外のURLを拒否」する作りなので、ここを自分のドメインにするのが安全上のキモです。

6

フロントを surge で公開

index.html と watch.html の入ったフォルダで：
surge ./ <あなたの名前>.surge.sh

7

運用：容量の管理

標準画質で2時間 ≈ 2.3GB → 10GB で約4本が目安。いっぱいになったら古いものから削除：

一覧：wrangler r2 object list <バケット名>
削除：wrangler r2 object delete "<バケット名>/rec/ファイル名"

⚠️

Worker本体のソースコード（合言葉の照合ロジックやR2バインドの詳細）は別ファイルです。④の中身まで「コピペで動く」完全版にしたい場合は、Worker のソース（~/WORKS/sakureco-storage/ 内のコードや wrangler.toml）も共有してください。
