Setup Manual

ポートフォリオサイトを
自分用に作って公開するマニュアル

プログラミング未経験でもOK。テキストを書き換えて、無料の surge.sh にアップロードするだけで、あなただけのポートフォリオサイトが完成します。

Overview

このサイトでできること

このテンプレートは HTML・CSS・JavaScript だけで作られた1ページ完結のポートフォリオサイトです。サーバーやデータベースは不要で、ファイルをアップロードするだけで動きます。

  • プロフィール・自己紹介・スキル・実績・お問い合わせを掲載できる
  • スマホ／タブレット／PCすべてに自動対応（レスポンシブ）
  • お問い合わせは Google フォームに飛ばすだけなので、サーバー処理が不要
  • 公開は surge.sh で完全無料。あなたの名前.surge.sh のようなURLがもらえる

ファイル構成

template/
├─ index.html   ← 文章・見出しを書き換えるメインファイル
├─ style.css    ← 色やデザイン（基本そのままでOK）
├─ script.js    ← アニメーションなどの動き（編集不要）
└─ images/      ← 写真・実績画像を入れるフォルダ
   ├─ profile.svg
   └─ project1.svg 〜 project6.svg

Requirements

準備するもの

  • テンプレート一式（template フォルダ）
  • テキストエディタ … VS Code がおすすめ（無料）。メモ帳でも可
  • Node.js … surge.sh で公開するために使います → nodejs.org から「LTS」版をインストール
  • プロフィール写真・実績画像（あとで差し替え可）
  • Googleアカウント … お問い合わせフォーム用（任意）
💡

Node.js を入れると npm というコマンドも一緒に入ります。これを使って公開ツール（surge）をインストールします。

Steps

サイトを自分用に編集する

1

テンプレートをコピーする

template フォルダを丸ごとコピーして、自分の作業用フォルダ（例：my-portfolio）として使いましょう。元データを残しておくと、やり直したいときに安心です。

2

index.html の文章を書き換える

index.html をエディタで開きます。<!-- ✏️ 編集: … --> というコメントが編集ポイントの目印です。下の表のプレースホルダ（仮の文字）を自分の情報に置き換えてください。

場所仮の文字（置き換える対象）入れる内容
タブのタイトルあなたの名前 | ポートフォリオサイト名・氏名
ロゴ／フッターYourName表示名（英字推奨）
ヒーローYour Title / キャッチコピー肩書き・キャッチコピー
バッジあなたの強みアピールしたい実績
Aboutあなたの名前 / 自己紹介文氏名・肩書き・経歴
スキルスキル1〜5 / 95%スキル名と習熟度
実績実績タイトル1〜6 など実績の内容・タグ
ツールツール1〜3使えるツール紹介
お問い合わせYOUR_GOOGLE_FORM_URLGoogleフォームのURL
⚠️

スキルバーの数値は2か所あります。<span class="skill-pct">95%</span> と style="width:95%" の両方を同じ数字にそろえてください。

3

実績の数を調整する

実績カードは <article class="work-card"> … </article> のかたまり1つ＝実績1件です。

  • 減らす：いらないカードを <article> から </article> まで丸ごと削除
  • 増やす：カード1つをコピーして貼り付け、中身を書き換える

ツールカード（tool-card）も同じ要領で増減できます。

4

画像を差し替える

images フォルダの中身を自分の画像に置き換えます。同じファイル名にすればそのまま反映され、別名にする場合は index.html の <img src="…"> を書き換えます。

<!-- 例：profile.svg を profile.jpg に変えた場合 -->
<img src="images/profile.jpg" alt="プロフィール写真" />
🖼️

写真は1枚 500KB以内 を目安に圧縮すると表示が速くなります。圧縮には Squoosh や TinyPNG が便利です。プロフィールは正方形、実績は横長（16:9）がきれいに収まります。

5

お問い合わせ用 Google フォームを用意する

  1. Googleフォーム で「名前・メール・お問い合わせ内容」などの項目を作る
  2. 右上の「送信」→ リンクアイコン（🔗）→ URLをコピー
  3. index.html の YOUR_GOOGLE_FORM_URL をそのURLに置き換える

フォームを使わない場合は、ボタンのリンク先をメール（mailto:あなた@example.com）やSNSのURLに変えてもOKです。

6

ローカルで表示を確認する

公開する前に、手元で見た目をチェックしましょう。index.html をダブルクリックしてブラウザで開くだけでもOKです。文章・画像・リンクが正しいか確認してください。

Deploy

surge.sh で無料公開する

準備ができたら、いよいよインターネットに公開します。ターミナル（Macは「ターミナル」、Windowsは「コマンドプロンプト」や「PowerShell」）を開いて、次のコマンドを順番に実行します。

① surge をインストール（最初の1回だけ）

npm install --global surge

② 公開するフォルダに移動する

# my-portfolio は自分の作業フォルダ名に置き換え
cd my-portfolio

③ 公開する

surge

初回はメールアドレスとパスワードを聞かれるので、入力するとアカウントが作られます。続いて公開先（ドメイン）を聞かれます。

project: /Users/you/my-portfolio/
domain:  好きな名前.surge.sh   ← ここを自分用に入力してEnter

「Success!」と表示されたら公開完了です。表示されたURL（例：https://好きな名前.surge.sh）にアクセスすると、あなたのサイトが世界中から見られます🎉

🔁

更新するときは、ファイルを編集したあと同じフォルダで surge を実行し、domain に同じドメイン名を入れるだけで上書き公開されます。
1行で済ませたいときは surge ./ 好きな名前.surge.sh でもOK。

⚠️

.surge.sh ドメインは早い者勝ちです。すでに使われている名前は選べません。別の名前を試してください。

Auto Expiry

3週間で自動的に「期限切れ」にする

このテンプレートには、公開から3週間が過ぎると自動でページを「期限切れ」表示に切り替える機能が入っています。期間限定の公開や、デモ用サイトに便利です。

設定方法

expiry.js の先頭にある公開日を、実際に公開した日に書き換えるだけです。

const PUBLISH_DATE = '2026-06-14'; ← 公開した日（YYYY-MM-DD）
const VALID_DAYS   = 21;           ← 公開期間。3週間=21日

期間を変えたいときは VALID_DAYS の数字を変更します（例：1週間なら 7）。

期限が過ぎると何が起こる？

  • ページの内容（個人情報を含む）が画面から自動的に削除される
  • ブラウザに保存されたデータ（localStorage・Cookie・キャッシュ等）も自動で消去される
  • 代わりに「このページの公開は終了しました」という画面が表示される
⚠️

この仕組みはブラウザ側（JavaScript）で動くため、サーバー上のファイルそのものは残ります。アップロードした元ファイルまで完全に消したい場合は、下のコマンドでサイトごと削除してください。

surge teardown 好きな名前.surge.sh
🤖

「3週間後に上のコマンドを自動で実行したい」場合は、GitHub Actions などの定期実行サービスを使えば完全自動化も可能です（やや上級者向け）。必要であれば スケジュール実行 を調べてみてください。

Customize

もっとカスタマイズしたい人へ

テーマカラーを変える

style.css の先頭にある :root の色コードを書き換えると、サイト全体の色が一括で変わります。

:root {
  --green: #2a9d5c;       ← メインカラー
  --green-light: #e8f5ee; ← 薄い背景色
  --green-dark: #1e7a45;  ← ホバー時の濃い色
}

好きな色は Coolors などで探せます。3色をまとめて変えると統一感が出ます。

FAQ

よくある質問・トラブル

Q. surge: command not found と出る

surge がインストールされていません。npm install --global surge を実行してください。それでもダメな場合は Node.js が入っているか node -v で確認します。

Q. 画像が表示されない

ファイル名の打ち間違い（大文字小文字も区別されます）か、images フォルダの外に画像がある可能性が高いです。src="images/ファイル名" とフォルダ・名前が一致しているか確認しましょう。

Q. 公開したサイトを消したい

surge teardown 好きな名前.surge.sh で削除できます。

Q. お金はかかる？

surge.sh の基本機能は無料です。Node.js・VS Code・Googleフォームもすべて無料で使えます。

Q. もっと本格的に運用したい

独自ドメインや GitHub 連携をしたい場合は、GitHub Pages や Netlify、Vercel も同じファイルのまま公開できます。
